# EdgeTap

[English](README.md)

**EdgeTap** 是一款原生 macOS 增强工具，将触控板边缘变为手势驱动的快捷操作区。无需改变手势握持，只需手指贴着触控板边缘滑动，即可实现音量调节、触发快捷键等功能。

## ✨ 核心特性

- **菜单栏常驻** — 超轻量后台运行，不占用 Dock 栏，所有设置整合在极简原生界面中。
- **连续音量滑轨** — 将任意边缘设为音量调节模式，沿边缘上下滑动即可无极调整系统音量，如同操控实体滚轮。
- **四边自定义快捷键** — 在顶部、底部、左侧、右侧四条边缘上，分别为上滑和下滑（或左滑和右滑）绑定独立的键盘快捷键组合。
- **四角点击** — 点击触控板四个角落（左上、右上、左下、右下）即可触发预设操作。
- **底层深度集成** — 接入 macOS 原生 `MultitouchSupport` 私有框架，直接读取电容阵列数据，手势识别精准且灵敏。

## 🚀 安装

### 方式一：从 Releases 下载

1. 前往 [Releases](https://github.com/kingwsi/EdgeTap/releases) 下载最新的 `EdgeTap-x.x.x.zip`。
2. 解压后将 `EdgeTap.app` 移动到「应用程序」文件夹。
3. 如果 macOS 阻止打开，请在终端执行：`xattr -cr /Applications/EdgeTap.app`

### 方式二：从源码编译

需要 Swift 5.9+ 和 macOS 13.0+。

```bash
git clone https://github.com/kingwsi/EdgeTap.git
cd EdgeTap
./build_app.sh
open EdgeTap.app
```

## 🛡 权限说明

EdgeTap 需要 **辅助功能** 权限来模拟按键操作。

首次启用时，应用会自动弹窗提醒。请前往：

**系统设置 → 隐私与安全性 → 辅助功能**

> 如果通过终端运行，请为你的终端程序（如 Terminal.app 或 iTerm2）授权。

## 🎮 使用方法

1. 点击菜单栏中的触控板图标 → **设置**。
2. 开启 **「启用 EdgeTap」** 总开关。
3. 切换标签页选择要配置的边缘（顶部、底部、左侧、右侧、角落）。
4. 对每条边缘选择模式：
   - **音量调节** — 沿边缘连续滑动，无极调节系统音量。
   - **自定义按键** — 为每个方向分别绑定键盘快捷键。

## 🏗 架构

| 层级 | 说明 |
|------|------|
| **EdgeTapApp** | 界面、设置中心、快捷键与媒体键模拟 |
| **EdgeTapCore** | 多点触控捕获引擎与手势识别系统（`EdgeSwipeDetector`） |
